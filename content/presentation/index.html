Présentation de ExtJS

ExtJS est une librairie Javascript cross-browser permettant de construire des applications internet riches, rapides, et évolutives.
• Performances élevées
• Design & ergonomie
• Haut degré de personalisation et de modularité
• Approche intuitive, API facile à prendre en main
• Supporté par les principaux navigateurs: Internet Explorer 6+, Firefox 1.5+, Safari 3+, Chrome 3+, Opera 9+
