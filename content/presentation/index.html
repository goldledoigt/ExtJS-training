Introduction
Présentation de ExtJS

ExtJS est une librairie Javascript cross-browser permettant de construire des applications internet riches, rapides, et évolutives.
• Performances élevées
• Design & ergonomie
• Haut degré de personalisation et de modularité
• Approche intuitive, API facile à prendre en main
• Supporté par les principaux navigateurs: Internet Explorer 6+, Firefox 1.5+, Safari 3+, Chrome 3+, Opera 9+

Ext Core

Ext Core est le coeur du framework Ext JS, il gère les fonctionnalités les plus basiques du developpement web: la manipulation des éléments du DOM, la gestion des évènements entre objets, les requètes Ajax, les templates HTML etc ...

Ext JS

Ext JS intègre toutes les fonctionnalités d'Ext Core en y ajoutant le concept de composants graphiques Ext.Component pouvant etre imbriqués, modifiés, surchargés...
