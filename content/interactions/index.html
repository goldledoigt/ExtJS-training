Manipulation des components
Interactions entre 2 componants

ExtJS permet de faire interagir plusieurs components facilement en créant un minimum de dépendances entre eux dans le code.
Dans l'exemple ci-dessous le FORM se rempli en fonction de la ligne sélectionnée dans le GRID.


Créer un évènement

L'utilisation des évènements ExtJS permet de faire communiquer deux composants ExtJS sans créer de dépendance entre eux dans le code. Les composants ExtJS on leur propres évènements référencés dans l'API, mais lors de leur surcharge il est possible dans créer de nouveaux.

Dans notre example le GRID déclenche un évènement lorsqu'une de ses ligne est sélectionnée. Sur le "handler" de cet évènement nous déclenchons à notre tour un autre évènement personnalisé: "edit".

,onRowSelect:function(selModel, rowIndex, record) {
	this.fireEvent("edit", record);
}


Relayer un évènement

Il est possible de relayer un évènement et de faire en sorte qu'un composant ExtJS écoute les évènements d'autre composants. Dans notre exemple l'évènement "edit" du GRID est relayer au FORM. 

form.relayEvents(grid, ["edit"]);


Et un "handler" de l'évènement "edit" est placé dans le FORM. 

this.on({
	edit:this.onEdit
});


Ainsi tout déclenchement de l'évènement "edit"dans le GRID appelera la méthode faisant office de "handler" dans le form.
